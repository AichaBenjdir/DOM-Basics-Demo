DOM Méthods

Le DOM (Document Object Model) est une interface qui permet de manipuler la structure, le style, et le contenu des documents web de manière dynamique.

Le DOM représente une page web sous la forme d'un arbre d'éléments, chaque balise HTML étant un nœud de cet arbre.
